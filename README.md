
# Two-way databinding with repeater controls

In this [Stadium](https://stadium.software) sample, we make use of the Repeater to illustrate how to how to implement data binding with repeaters that allows 2-way binding to items in a dataset. The updated dataset can be easily read for further use.

#### Sample Highlights:

- Assign types to repeaters
- Databind controls inside the repeater using ListItem Type
- Using/reading updated dataset items

---
#### Using the Sample application:
1. Download the Stadium Designer
2. Download the sapz file in this repository
3. Open the sapz inside of the Stadium Designer
4. Click Preview
